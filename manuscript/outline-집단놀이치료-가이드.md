# 집필 개요: 집단 놀이 치료 실전 가이드 (Gems 기획안 기준)

## 한 줄

> 이 책은 **학교·시설에서 5~10회기 단기 집단 놀이 치료**를 설계·진행하는 실무자가, **Yalom·Winnicott·Kottman·LEAP**을 놀이에 연결해 **라포·안전·행동 변화**를 만들도록 돕는다.

## 작가/톤 (Gems 시스템 지시 요약)

- 역할: 파란나라 현장 중심, 기독교 가치 + 비기독교 환경에서도 쓸 수 있는 전문 문체.
- 매 회기: **Warm-up → Main → LEAP** 필수.
- 이론 뒤: **파란나라 현장 Tip** / **돌발 Q&A** 필수.
- 사례: 문제 행동 → 개입 → 역동 변화 → 치료사 성찰.

## 목차 골격 (제1~5부)

1. 놀이 치료의 기초와 집단 역동 (Yalom 11요인, Winnicott, 아동중심·인지행동 병행)
2. 회기 황금 구조 (도입·전개·LEAP) + 집단 발달 단계
3. 매체별 기법 (보드게임, Bibliotherapy, 미술·공예) + 실전 팁
4. 5~10회기 커리큘럼 (안아주기 → 참자기 → 홀로서기)
5. 한계 설정·돌발·기록·윤리·동의
