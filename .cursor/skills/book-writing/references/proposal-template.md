# 출판사 투고 기획안 템플릿

## 기획안 작성의 목적

출판사 편집자는 하루 수십 개의 투고를 받는다. 기획안은 "이 책을 왜 지금, 왜 이 저자가, 왜 당신 출판사에서 내야 하는지"를 설득하는 문서다.

## 기획안 작성 팁

### 좋은 한 줄 소개의 예시

- "지친 그리스도인을 위한 영적 피로 회복 안내서"
- "침례교 목사가 30년 목회에서 배운 가정 예배의 모든 것"
- "NGO 현장에서 만난 하나님의 얼굴 — 고통 받는 이웃에게서 발견한 복음"

### 경쟁 도서 분석 방법

1. 교보문고에서 같은 키워드로 검색
2. 베스트셀러 10위 내 유사 도서 파악
3. "나는 이 책과 이런 점이 다르다"를 명확히

### 저자 플랫폼이 없을 때

출판사는 판매 가능성을 본다. 플랫폼이 없다면:
- 교회 성도 수 / 예상 구매 인원 명시
- "교회 내 1차 배포 ___권 예정" 언급
- 강연·집회 청중 규모 기재
- 출판 후 홍보 계획 구체적으로 제시

## 투고 이메일 샘플

```
제목: [기획안 투고] 책 제목 / 저자명

안녕하세요, [출판사명] 편집부 담당자님.

저는 [지역] [교회명]에서 [__년째] 사역하고 있는 
[이름] 목사입니다.

[책에 대한 한두 문장 소개]

이 책이 [출판사명]의 독자들과 잘 맞을 것 같아 
기획안을 보내드립니다. 첨부 파일에 기획안과 
샘플 챕터 2편을 담았습니다.

검토 후 연락 주시면 감사하겠습니다.

[이름] 드림
[연락처]
```
