# 간증문 & 자서전 집필 가이드

## 간증문·자서전의 힘

간증은 기독교 문학의 가장 오래된 장르다. 아우구스티누스의 『고백록』부터 오늘의 간증집까지 — 모두 같은 구조를 가진다:

> **나는 이랬다 → 하나님을 만났다 → 나는 달라졌다**

그러나 독자에게 진짜 힘을 주는 간증은 단순한 **성공 스토리**가 아니다. 취약함과 솔직함이 있는 이야기가 독자의 마음을 건드린다.

---

## 간증문의 구조

### 기본 3막 구조

```
1막: 이전 (Before)
  - 나는 누구였는가?
  - 어떤 문제/고통/한계가 있었는가?
  - 내가 시도했지만 실패한 것들

2막: 전환 (Turning Point)
  - 만남의 순간 (하나님 / 사건 / 사람)
  - 내면의 싸움과 항복
  - 결정적 변화의 순간

3막: 이후 (After)
  - 무엇이 달라졌는가?
  - 아직 진행 중인 여정
  - 독자에게 전하고 싶은 소망
```

### 좋은 간증문의 특징

1. **솔직한 취약성**: "저는 이 시기에 하나님을 원망했습니다"
2. **구체적 세부 묘사**: 날짜, 장소, 감정, 대화를 구체적으로
3. **과장 없음**: 기적을 과장하면 오히려 신뢰를 잃는다
4. **현재 진행형**: "완전히 다 해결됐다"보다 "지금도 이 여정 중에 있다"

## 자서전 집필 순서

### 1단계: 인생 타임라인 작성

```
출생 → 어린 시절 → 청소년기 → 청년기 →
사역/직업 시작 → 주요 사건들 → 현재
```

각 단계에서 뽑을 것:
- 가장 기억에 남는 장면 1~2개
- 인생을 바꾼 결정 1~2개
- 하나님의 손길이 보이는 순간들

### 2단계: 이야기할 챕터 선별

모든 것을 다 쓰려 하면 아무것도 전달되지 않는다. **10~15개의 핵심 이야기**를 선별한 후 각각 깊이 있게 쓴다.

### 3단계: 비선형 구성 고려

반드시 연대기 순서일 필요는 없다:

```
[주제별 구성 예시]
1부: 기도에 대하여 (기도 관련 사건들 모음)
2부: 실패에 대하여 (실패와 회복 사건들)
3부: 사람에 대하여 (관계 속 하나님의 섭리)
```

## 간증·자서전 집필 시 주의점

### 타인에 대한 서술
- 살아 있는 실명 인물을 부정적으로 묘사할 때 주의
- 가능하면 이름을 바꾸거나 익명 처리
- "이 글을 먼저 보여줄 것인가?" 미리 판단

### 과거 죄/실수 서술
- 솔직하게 쓰되 불필요한 세부 묘사는 피함
- 목적: 고백과 용서의 은혜를 나누는 것
- 가족이 읽었을 때 상처받지 않을 수준 고려

### 하나님 음성/환상/기적 서술
- 신중하게, 검증 가능한 수준에서 서술
- "이것이 하나님의 음성이었다"보다 "나는 그것이 하나님께서 주신 마음이라 믿었다"
