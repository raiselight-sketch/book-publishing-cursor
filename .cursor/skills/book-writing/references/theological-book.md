# 신학서 & 설교집 집필 가이드

## 신학서/설교집의 특수성

일반 책과 달리 신학서·설교집은 두 가지 독자를 동시에 만족시켜야 한다:
1. **신앙인 독자**: 감동, 적용, 실천 동기
2. **목회자/신학생 독자**: 신학적 깊이, 본문 충실성, 학문적 신뢰

---

## 설교집 편집 지침

### 설교 → 글로 변환할 때 반드시 할 것

설교는 **듣는** 것이고, 책은 **읽는** 것이다. 이 차이를 무시하면 책이 되지 않는다.

```
[설교의 특성]          [책의 특성]
반복이 많음      →    군더더기 제거
구어체           →    문어체로 전환 (단, 온기는 살림)
몸짓/표정 있음   →    언어만으로 표현
청중 반응 있음   →    독자 반응을 글로 유도
시간 제한 있음   →    깊이 확장 가능
```

**전환 4단계:**
1. 설교 녹취록 또는 원고 확보
2. 구어체 표현 → 문어체 변환
3. 반복 단락 통합, 불필요한 감탄사 제거
4. 독자를 위한 적용 섹션 추가 (설교엔 없던 부분)

### 설교집 목차 구성 패턴

**패턴 A: 강해 시리즈형**
```
창세기 강해 1부 (1:1~2:25) — 창조의 하나님
  1. 태초에 (창 1:1~2)
  2. 빛이 있으라 (창 1:3~5)
  ...
```

**패턴 B: 주제별 묶음형**
```
1부 — 기도의 사람들
  1. 아브라함의 기도
  2. 한나의 기도
2부 — 고난의 사람들
  ...
```

**패턴 C: 절기/시즌형**
```
대림절 설교 (4편)
부활절 설교 (4편)
추수감사절 설교 (2편)
...
```

## 신앙 에세이 집필 원칙

### 신앙 에세이의 황금 공식

```
개인 경험 (40%) + 성경 진리 (40%) + 독자 초대 (20%)
```

- **개인 경험**: 구체적, 취약하고 솔직하게. 완벽한 목사 이미지 내려놓기
- **성경 진리**: 설명이 아닌 발견으로. "이런 뜻입니다"보다 "이걸 발견했을 때…"
- **독자 초대**: "당신은 어떤가요?" 독자 자신의 이야기로 연결

### 피해야 할 신앙 에세이의 함정

1. **도덕주의**: "이렇게 해야 합니다" 반복 → 독자가 지침
2. **승리주의**: 항상 이기고 극복하는 이야기 → 공감 불가
3. **신학 강의**: 에세이가 설교가 되어버림
4. **추상 언어**: "사랑", "은혜", "믿음"만 반복 → 구체화 필요

## 신학서 학문적 장치

### 각주/미주 처리
- 인용문은 반드시 출처 표기
- 형식: 저자명, 『책 제목』(출판지: 출판사, 연도), 페이지.
- 성경 인용: (개역개정) (새번역) 등 번역본 명시

### 참고문헌 목록
- 국내 문헌 먼저, 이후 외국 문헌
- 외국 문헌은 원어 표기 (영어, 독어 등)
- 알파벳 / 가나다 순 정렬

### 성경 인용 원칙
- 기본: 개역개정
- 필요 시 다른 번역 병기: "새번역에서는 … 로 번역한다"
- 히브리어/헬라어 원어 사용 시: 음역 병기 권장

## 신학서 추천 구조

### 단행본 신학서 (예: 기도론)

```
프롤로그: 저자의 기도 여정
1장: 성경에서 기도란 무엇인가 (정의/본질)
2장: 구약에서 배우는 기도 (역사·사례)
3장: 예수님의 기도 신학 (복음서)
4장: 바울 서신의 기도론 (서신서)
5장: 기도의 신학적 도전들 (현대적 질문)
6장: 기도의 실천 (적용)
에필로그: 기도하는 교회를 향하여
부록: 기도 용어 해설 / 참고문헌
```

## 큐티(QT)집 / 묵상집 집필

### 하루 묵상 단위 구조

```
제목 (한 줄)
본문 (3~10절)
---
[여는 이야기] — 일상의 공감되는 짧은 이야기 (3~5문장)
[본문 묵상] — 본문의 핵심 메시지 해설 (5~8문장)
[적용 질문] — 오늘 나의 삶에 적용할 1~2가지 질문
[기도] — 짧은 기도문 (3~5문장)
---
암송 구절 (선택)
```

### QT집 분량 가이드

```
1일 1편 × 30일 = 30편 (한 달치)
1편 분량: 500~800자
총 원고량: 약 2~3만 자
```
